Support

Use this page as the support destination for the App Store listing and GitHub Pages launch site.

Common Setup Steps

  • Open SucceedAI from the macOS menu bar.
  • Open Settings and verify Accessibility permission is enabled.
  • Check that your replacement trigger is correct in Settings > Keys.
  • Type the trigger in an editable text field, describe the task, and press Return.

Need Help?

Open a GitHub issue with your macOS version, app version, the app where the replacement failed, and whether Accessibility permission is enabled.
