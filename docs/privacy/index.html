Privacy Policy

Last updated: May 7, 2026

SucceedAI is designed to provide in-place AI writing assistance on macOS while keeping the app workflow simple and transparent.

Information Processed

When you use the AI replacement feature, the text you ask SucceedAI to process is sent to the configured SucceedAI backend proxy so the request can be completed by the configured AI provider.

macOS Permissions

SucceedAI requests Accessibility permission because macOS requires it for apps that detect global keyboard input and insert generated text into the active app. This permission is used for trigger detection and response insertion.

Backend Proxy

The app connects to the configured SucceedAI proxy API. AI provider API keys are stored on the backend, not entered into the public app interface.

Data Retention

The app should not store your AI prompts locally beyond the active request flow. Backend logging should avoid storing user prompt content unless explicitly enabled for debugging.

Contact

For privacy questions, contact support through the support page.
