## How It Works

H2:
One trigger. One prompt. One Return key.

Steps:
1. Open any app with an editable text field.
2. Type your SucceedAI trigger and describe the task.
3. Press Return.
4. SucceedAI replaces the command with the generated response.

Example:
`/ai rewrite this reply so it sounds warmer and more concise`

## Permissions

H2:
Clear macOS permissions, explained before you grant them

Body:
SucceedAI needs Accessibility permission because macOS requires it for apps that detect global keyboard input and insert text into other apps. The app explains the reason, opens the correct System Settings pane, and lets you check the permission status from Settings.

## FAQ

### What is SucceedAI?

SucceedAI is an AI writing assistant for Mac that replaces typed commands with AI-generated text directly inside the app you are using.

### Does SucceedAI work in any Mac app?

SucceedAI is designed for editable macOS text fields, including common writing surfaces like Mail, Notes, browser text fields, messaging apps, and documents.

### Why does SucceedAI need Accessibility permission?

macOS requires Accessibility permission for apps that detect global keyboard input and insert text into other apps. SucceedAI uses this permission for command detection and response insertion.

### Can I change the command trigger?

Yes. The Settings > Keys panel lets you change the replacement trigger and restore the default.

### Is SucceedAI a text expander?

It behaves like an AI-powered text replacement tool. Instead of expanding a static snippet, it sends your instruction to the configured AI service and inserts the generated response.

### How is it different from ChatGPT in a browser?

SucceedAI is optimized for in-place writing. You do not need to leave the app, paste text into a browser, copy the answer, and return to your original editor.
