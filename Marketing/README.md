# SucceedAI Marketing Launch Kit

This folder contains the acquisition assets for SucceedAI: SEO copy, Product Hunt launch copy, App Store metadata, landing page content, outreach templates, press copy, and launch checklists.

## Positioning

SucceedAI is an AI text replacement assistant for macOS. Users type a configurable command trigger in any editable Mac text field, describe what they want, press Return, and SucceedAI replaces the command with AI-generated text in place.

## Core Promise

Write with AI without switching apps.

## Primary Audience

- Mac professionals who write emails, notes, support replies, documentation, and messages all day.
- Founders, marketers, creators, operators, developers, support teams, recruiters, consultants, and students.
- Users who already use ChatGPT-like tools but dislike copy-paste loops and context switching.

## Acquisition Surfaces

- Product Hunt: launch listing, gallery, maker comment, launch replies.
- App Store: Mac screenshots, subtitle, keyword field, description, promotional text.
- Organic SEO: landing page, FAQ, use-case pages, structured data, image alt text.
- Social: X, LinkedIn, Bluesky, Reddit-style founder posts.
- Direct outreach: creator, productivity, Mac app, and AI newsletter pitches.
- GitHub: clear README with keywords, screenshots, trust, and build instructions.
