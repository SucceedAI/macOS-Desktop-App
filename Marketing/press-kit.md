# Press Kit

## One-Liner

SucceedAI is an AI text replacement assistant for macOS that lets users write with AI inside any editable Mac text field.

## Short Boilerplate

SucceedAI is a macOS menu bar app for AI-powered text replacement. Users type a configurable command trigger in any editable text field, describe what they need, press Return, and replace the command with generated text in place. It helps Mac users rewrite emails, summarize notes, draft replies, translate messages, and polish writing without switching apps.

## Longer Boilerplate

SucceedAI is built for Mac users who already use AI for everyday writing but dislike the copy-paste loop created by browser chat tools. Instead of moving text between apps, users can type a command directly where they are writing. SucceedAI detects the command, sends the instruction to the configured AI service, and inserts the generated response back into the active text field.

The app focuses on a small set of native macOS behaviors: a menu bar control center, guided Accessibility permission setup, configurable replacement trigger, and direct in-place text insertion.

## Founder Story Angle

AI writing is useful, but the workflow is often awkward. For small writing tasks, opening a separate AI window can take longer than the task itself. SucceedAI started from the idea that AI should behave like a native writing action on Mac: type a command, press Return, and continue where you already are.

## FAQ

### What problem does SucceedAI solve?

It removes the copy-paste loop from AI writing on Mac.

### Who is it for?

Mac users who write frequently: founders, marketers, operators, developers, support teams, consultants, students, creators, and knowledge workers.

### Why is it different?

It is not a destination app. It is an in-place writing action triggered from the text field the user is already editing.

### Why does it need Accessibility permission?

macOS requires Accessibility permission for global keyboard detection and text insertion into other apps.

### What are common workflows?

Rewrite, summarize, translate, shorten, expand, polish, draft replies, clean notes, and generate quick copy.
